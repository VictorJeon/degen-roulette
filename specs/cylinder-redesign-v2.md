## 레퍼런스 이미지 분석 (정확히 따라야 함)

### 1. 외곽 형태 — 기어/톱니 모양
- **현재 문제**: 원형 + 작은 원으로 노치를 표현 → 레퍼런스와 전혀 다름
- **레퍼런스**: 6개의 **크고 뚜렷한 직사각형/사다리꼴 톱니**가 원 외곽에 돌출
- 톱니는 챔버와 챔버 **사이** 위치 (30° 오프셋)
- 톱니 높이: 반지름의 ~8-10% 돌출
- 톱니 폭: ~20° 아크
- 톱니 꼭대기는 평평(flat top), 모서리는 약간 챔퍼(45° 베벨)
- **구현**: `<path>` 또는 `<polygon>`으로 톱니 포함한 단일 외곽선 그리기. 원 + 별도 원 조합 X.

### 2. 외곽선 + 글로우
- 외곽선 색: `#BFFF00` (네온 라임)
- 스트로크 폭: 2.5-3px
- 글로우: `feGaussianBlur` stdDeviation 4 + 8 이중 레이어
- 외곽선과 본체 사이에 약간의 간격 느낌 (내부 본체는 외곽보다 2-3px 작게)

### 3. 본체 색상
- 다크 올리브/밀리터리 그린: `#2a3a2a` → `#1a2a1a` → `#0d1a0d`
- radialGradient, 중앙 약간 밝고 가장자리 어둡게
- 표면에 미세한 텍스처감 (fluting line 유지하되 더 미묘하게, opacity 0.3)

### 4. 챔버 홀 (핵심 차이!)
- **레퍼런스**: 각 챔버에 **2중 링** 구조
  - 외곽 링: 밝은 메탈릭 그린/골드 (`#7a8a5a` ~ `#a0b070`), 폭 3-4px
  - 내부 링: 더 어두운 메탈릭 (`#3a4a3a`), 폭 2px
  - 가장 안쪽: 깊은 검정 (`#050505`)
- 입체감을 위해 외곽 링 상단에 하이라이트 (`stop-color: #a0b070` at 20%)
- 챔버 반지름: 현재 r=24 → r=26으로 약간 키우기
- 외곽 링은 r=28, 내부 링은 r=22

### 5. 중앙 허브
- **레퍼런스**: 눈에 띄는 원형 허브 + 내부에 `+` (십자) 아이콘
- 허브 외곽 링: `#4a6a3a` 메탈릭 그린, r=18, 스트로크 2px
- 허브 내부: `#2a4a2a` 채움
- 십자 아이콘: `#BFFF00` 색상, 스트로크 2px, 길이 ±8px
- 허브에도 미세한 글로우 (hubGlow 필터)

### 6. 프레임
- `.revolver-frame`의 `border-radius: 20%` 유지
- 네온 글로우 box-shadow 유지
- 배경은 반투명 다크 (`rgba(5, 15, 5, 0.7)`)

## 동작 요구사항 (변경 X)
- 챔버 선택, 회전 애니메이션, 총알 표시 등 기존 로직 그대로
- `canSelect` 시 챔버 외곽 링이 `#BFFF00`으로 하이라이트
- fired 챔버는 빨간색 유지

## 검증
1. `npx tsc --noEmit` 통과
2. `npm run build` 성공
3. 시각적으로 레퍼런스와 유사한지 확인 (톱니 형태가 핵심)

## 핵심 원칙
- **SVG path로 톱니 외곽선 구현** — circle + 별도 도형 조합이 아닌, 톱니가 포함된 단일 path
- **챔버는 2중 링** — 단순 원 2개가 아닌 메탈릭 그라데이션 링
- **레퍼런스의 3D/입체감** 최대한 살리기 (그라데이션 + 하이라이트)
