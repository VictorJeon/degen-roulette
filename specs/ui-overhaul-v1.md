## 개요
현재 단일 컬럼 모바일 중심 UI → **3컬럼 데스크톱 게이밍 대시보드**로 전환.
전체적인 네온 사이버펑크 톤 강화. 미드저니 레퍼런스를 최대한 따르되, 현실적으로 구현 가능한 선에서.

## 파일 구조
- `frontend/components/GameBoard.tsx` — 메인 게임 컴포넌트 (대폭 수정)
- `frontend/app/globals.css` — 전역 스타일 (대폭 수정)
- `frontend/app/page.tsx` — 페이지 레이아웃 (3컬럼 구조로 변경)
- 필요시 새 컴포넌트 파일 생성 가능

## 1. 레이아웃 (3컬럼)

```
┌──────────────────────────────────────────────────────┐
│                    TOP BAR (고정)                      │
├──────────┬───────────────────────────┬───────────────┤
│  LEFT    │        CENTER             │    RIGHT      │
│  ~18%    │        ~52%               │    ~30%       │
│  Hall of │     Game Area             │  Live Feed    │

│  +       │                           │               │
```

- CSS Grid: `grid-template-columns: 230px 1fr 380px`
- gap: 8-12px
- 모바일 대응은 이번 작업에서 안 함 (데스크톱 우선)
- min-height: 100vh, 배경 다크

## 3. 탑바

왼→오: `DEGEN ROULETTE` 로고 | 중앙: 결과 배너 (게임 중일 때 `RESULT: SAFE` 등) | 오: `Total Plays: 12,647` + 지갑 연결 버튼

- 배경: 반투명 다크 + 하단 보더 (`border-bottom: 1px solid rgba(191, 255, 0, 0.2)`)
- 결과 배너: 둥근 알약 형태, 네온 보더, 텍스트 글로우
- 지갑 버튼: 연결 시 지갑 주소 축약 표시 (`9FPY...NJWx`)

## 4. 왼쪽 사이드바 — Hall of Degens

현재 `HallOfDegens` 컴포넌트를 사이드바로 이동.

상단 섹션: **HALL OF DEGENS** (타이틀)
- 리더보드 리스트: 지갑주소 축약 + 수익 (+0.099, -0.010 등)
- 양수: 녹색(`#00FF41`), 음수: 빨간색(`#FF0040`)

하단 섹션: **ORBIT NIGS ◇ RANK** (또는 적절한 이름)
- 별도 랭킹 리스트

스타일:
- 배경: `rgba(10, 20, 10, 0.7)` + 왼쪽 보더 네온 라인
- 폰트: 픽셀 폰트 (기존 유지)
- 항목 간격: 작게 (compact)

## 5. 중앙 게임 영역

### 5.1 "YOU LIVE." 텍스트
- 게임 결과 시 큰 텍스트: `YOU LIVE.` (생존) / `YOU DIED.` (사망)
- 색상: 생존=`#00FF41`, 사망=`#FF0040`
- 글로우 + 큰 폰트 (2-3rem)
- 서브텍스트: `1 BULLET. 6 CHAMBERS. HOW DEGEN ARE YOU?`

### 5.2 멀티플라이어 테이블
- **가로 배치** (현재도 가로지만 더 강조)
- 현재 활성 라운드: 밝은 배경 + 흰 텍스트, 나머지는 어둡게
- R3 활성 예시: `R3 | 1.94x` 부분만 밝은 배경
- 아래에 작은 텍스트로 승률 표시: `3/1`, `5/6`, `MAX 5/6`, `1.22x`, `1/8`, `1/6`

### 5.3 실린더 영역
- 현재 PNG 에셋 방식 유지
- 주변에 네온 글로우/스파클 효과 추가 (CSS로)
- 실린더 주변에 원형 네온 링 (CSS box-shadow 또는 border)

### 5.4 베팅 패널
- `>>> SELECT YOUR BET <<<` 텍스트
- 입력 필드: 네온 보더, 안에 값 + 화살표 아이콘 + `YAE` 텍스트 (또는 SOL)
- 프리셋 버튼: `0.005 | 0.001 | 0.013 | 0.015 | 0.03` — 선택된 건 밝은 네온 배경
- 그 아래: `Potential Payout: 0.036` / `Potential Loss: 0.01 SOL`
- **PLAY AGAIN** 버튼: 큰 사이즈, 네온 그린 보더, 텍스트 글로우

## 6. 오른쪽 사이드바 — Live Feed

타이틀: **LIVE FEED** + 우측에 `GLOBAL` 텍스트

탭: `GLOBAL` (활성, 체크마크 아이콘) | `FRIENDS`

각 항목:
```
✅ GLOBAL · FRIENDS
9FPY...NJWX · R3 · 0.01 SOL
SAFE · #01          +0.019
```

- SAFE = 녹색 배지, BANG = 빨간 배지
- 각 항목에 얇은 하단 구분선
- 스크롤 가능

## 7. 글로우/이펙트

- 실린더 주변: 방사형 녹색 글로우 (`box-shadow: 0 0 40px rgba(0, 255, 65, 0.3)`)
- 스파클: CSS animation으로 작은 반짝이 효과 (과하지 않게)
- 모든 패널: 미세한 네온 보더 글로우
- 배경: 미세한 그리드/스캔라인 패턴 (선택적)

## 8. 타이포그래피

- 메인: 기존 픽셀 폰트 유지
- 결과 텍스트/제목: 더 큰 사이즈 + letter-spacing
- 숫자: 고정폭

## 주의사항
- **실린더는 현재 PNG 에셋 방식 유지** — SVG로 다시 바꾸지 말 것
- **게임 로직(useGame.ts, API 등)은 건드리지 말 것** — 순수 UI/CSS 작업
- **모바일 대응은 이번에 안 함** — 데스크톱 1200px+ 기준
- 레퍼런스는 미드저니 이미지라 텍스트가 뭉개져 있음 — 텍스트 내용은 현재 코드 기준 유지
- 레퍼런스의 과한 부분은 적절히 절제 (글로우 과다, 스파클 과다 등)
